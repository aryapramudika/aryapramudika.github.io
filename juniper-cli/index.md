# Basic Juniper CLI

## Konfigurasi Hostname

```Shell
root@Juniper-R1# set system host-name Juniper-R2
root@Juniper-R1# commit 
root@Juniper-R2#
```

## Konfigurasi Root Password

```Shell
root@Juniper-R2# set system root-authentication plain-text-password    
New password:
Retype new password:
root@Juniper-R2#
```
